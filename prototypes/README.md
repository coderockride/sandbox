## Quick preview options

- Open the file directly:
  - macOS: `open prototypes/multiplier.html`
  - Linux: `xdg-open prototypes/multiplier.html`
  - Windows (PowerShell): `start prototypes/multiplier.html`
- Or drag `prototypes/multiplier.html` into an existing browser window.

## Optional local server

If you prefer serving over HTTP:

```bash
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000/prototypes/multiplier.html`.
